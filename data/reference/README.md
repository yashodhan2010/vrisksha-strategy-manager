# Nifty 500 Universe

Excel is the editable source of truth for the strategy universe.

1. Copy `nifty500_universe.example.xlsx` to `nifty500_universe.xlsx`.
2. Replace the fictional example rows with real Nifty 500 data.
3. Run `python -m app.main sync-universe`.

The generated `nifty500_universe.json` is the validated runtime artifact.
